https://{{ SITE_DOMAIN }}/api/getNumber?api_key=$api_key&service=$service&country=$country&pool=$pool&max_price=$max_price&areas=$areas

Parameters
&action=getNumber
$api_key — your API key;
$service — service code;
$pool - pool of number to order from
$max_price - the maximum price you're willing to pay for the number
$areas - preferred area codes

Possible mistakes
 NO_MONEY — Not enough balance left;
NO_NUMBERS - repeat a request or choose another country;
BAD_SERVICE - incorrect service name;
BAD_COUNTRY - incorrect country name;
NO_POOL - preferred pool to order from;
BAD_KEY - check your API key;

https://{{ SITE_DOMAIN }}/api/setStatus?api_key=$api_key&status=$status&id=$id

Parameters
$api_key — your API key;
$id — activation id;
$status - activation status;
3 — wait for the next code on the same number;
6 — complete activation;
8 — cancel activation;

Possible mistakes
 NO_ACTIVATION — the activation id does not exist;
BAD_KEY - check your API key;
BAD_STATUS — incorrect status EARLY_CANCEL_DENIED - It is possible to cancel the number after 2 minutes following the purchase;

The example of successful answer: 

{
  "id":2427,
  "phone":"+62895320077478",
  "pool":"Charlie",
  "product":"Whatsapp",
  "price":1954.55,
  "status":"Received",
  "expires":"2024-08-23 22:11:39",
  "sms":null,
  "created_at":"2024-08-23 21:51:39",
  "country":"Indonesia"
}

Response After Successful Number Cancellation
 ACCESS_CANCEL - activation canceled
